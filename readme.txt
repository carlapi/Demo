crea una cartella sul desktop e right click per aprire la banch e poi scrivi git config...
poi right click nella cartella e 
git init here
inizializza il git e ora git keeps track di tutto quello che succede nella cartella e solo la' !

e se cancelli la .git allora elimina tutto e scompare tutta la cronologia
